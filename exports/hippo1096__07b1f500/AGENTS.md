# AGENTS.md — 工作区规则

## 任务冲突

收到任务后先读 `.hermes/task-registry.json`，检查活跃任务。
与活跃任务冲突的写入操作 → 暂缓并告知用户。
行为规则详见 `~/.hermes/SOUL.md`。

## 文件落点

| 类型 | 位置 |
|------|------|
| 正式项目 | `./MMDDNN-项目名/` |
| 项目内临时/缓存 | `…/项目名/_work/` |
| 小任务临时文件 | `D:/tmp/YYYYMMDD-task-slug/` |
| 用户直接看的成品 | 项目根目录 |

禁止在 `D:\` 裸根目录散放文件。

## 项目目录规范

```text
001原始/      ← 不可变输入
002过程/      ← 分析报告、计划
_work/        ← 可重建的中间产物
```

生命周期：活跃 → `00归档/`（溯源保留）→ `00垃圾箱/`（可删除）。

## 语言

全中文交互。文档、表格、日志用中文编写，仅保留技术保留字。
